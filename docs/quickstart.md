# Quickstart

## 1. Clone

git clone <your-fork>

## 2. Add a project

Create:

context/projects/<project-id>/

Add:
- project_profile.md
- current_snapshot.md
- handoffs/
- feedback/

## 3. Register project

Add registry file with:

- project_id
- source_repo

## 4. Run a task

1. write a handoff
2. let agent execute
3. ensure feedback is written

If no feedback exists, the project is not yet activated.
